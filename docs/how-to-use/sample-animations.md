# Sample Animations

Some examples of the kinds of animations you can create with Keyframe Toolbox. Think of this is a cheat sheet that links the kinds of instructions you might get from a client with the kinds of curves you'd need to make it happen.

All these videos can be played at full size, and you'll see the playhead move through the graphs as each animation plays.
